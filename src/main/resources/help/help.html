Main Menu

Figure 1: Main menu

When the application first loads, you will be directed to the Main menu pane (shown in figure 1 above). This is a menu containing all the functions of the NameSayer app. From left to right, we have buttons to access the:

1)    Practice menu

2)    Listen menu

3)    Record menu

4)    Help menu

5)    Shop menu

All these menus will be further explored below.

To start, try using the Record menu to record a new name fragment into the database. To access this menu, simply click the Record menu button.

 

Record Menu

Figure 2: Record menu

The Record menu is a place to add recordings directly to the database. When you type in a name that you want to practice in the Practice menu, the application will piece together different name fragments from the database to form the chosen name.

Recording a new name

To record a new name into the database, simply follow these instructions:

1)     Enter the name you wish to record in the given text box. This should preferably be no more than one word in length.

2)     Check your microphone status with the microphone bar. Tapping on your microphone should make it spike.

3)     Press the Record button to begin recording a new name. The recording will start immediately, so be ready!

 

As you are recording the new name, the Record menu should now display a timer of seconds remaining, as well as change the Record button.

Figure 3: Recoding menu during recording

To finish and check your recording, you can do the following steps:

4)     Press the Stop Recording button to stop the recording at any time, otherwise, you can let it time out.

5)     Press the Play button to listen to your current recording of the name. If the quality is unsatisfactory, you may re-record the name by pressing the Record button once more.

6)     Press the Save button to store the recording in the database once you are satisfied with it. You can listen to it in the Listen menu.

7)     Press the Back button to go back to the Main menu. You can do this at any time. Any recording that you have made will not be stored in the database, but it will remain temporarily until you close the program.

Figure 4: Listen menu, Database tab

The Listen menu displays all the names in the database, as well as all the attempted names that the user has saved while practising. The Listen menu is divided into two tabs, which displays each type of recording respectively.

Database names

For Database names, like the one you have just recorded, as well as the many that come pre-packaged with the program, you can do the following:

1)     Play the recording by pressing the Play button. Sometimes, people will record full names instead of just one name (aka. a name fragment). This is unsatisfactory and should be disliked.

2)     Dislike a recording by pressing the Dislike button. Once a recording is disliked, the Dislike button will appear orange. Disliked recordings will only be used in practices if there is no other alternative. A list of all recordings which have been marked as disliked is saved as a text file called bad_quality.txt in the current working directory.

 

Attempted names

You can switch between tabs on the left-hand side of the screen. If you switch to the Attempted Names tab, you should see an interface like this:

Figure 5: Listen menu, Attempts tab

For Attempted names, which are recorded during Practice mode, you may click the icon to expand a name for more options.

Figure 6: Listen menu, Attempts tab, expanded

After a name has been expanded, you should see a list of all your attempted recordings for that name, numbered sequentially.

Just like before, you can:

1)     Play the recording of the attempted name by pressing the Play button.

2)     Delete the recording of the attempted name by pressing on the Delete button. This will show you a confirmation dialog to confirm your delete.

In both menus, you may also search for recordings with the Search bar. This will search case insensitively for all occurrences of the word, wherever it is.

Figure 7: Searching in the Listen menu

As with all the other menus, you can return to the Main menu at any time by pressing the Back button, on the top-left corner of the screen.

 

Practice Menu

Figure 8: Practice menu

The Practice menu is used for practising different names of people, and its the main functionality of our program.

To practice a list of names, you can do the following:

1)     Enter the name you wish to practice. The text field also supports autocomplete, which makes spelling the names a lot easier. Once you have typed in a name to practice, you can:

2)     Add the typed name to the practice list by pressing the Add button. This will simply add your typed name to the list of names you are going to be practising.

If the program cannot make your name by combining name fragments in the database, it will display an error. If you wish to practice that name, you can easily just go to the Recording menu and record whatever part of that name that wasnt in the database as a new name fragment.

 

Otherwise, you may import a list of names from your file system:

Figure 9: Importing files in the Practice menu

3)     Press the File button to import the names from a .txt file anywhere on your computer. Each name should be on a new line. If the program detects any names that it cannot generate by combining names in the database, it will again alert you and give you an error.

Once you have a list of names you wish to practise, it may look like this:

Figure 10: Practice menu with a list of names

If you have accidentally added a name, you can remove it by selecting it and clicking the Remove selected button. If you wish to clear the entire list, click Remove all.

You may choose to shuffle the order of the names randomly by checking the Shuffle checkbox. Once you are ready, hit the Start button to continue.

As always, you can return to the Main menu by clicking the Back button at the top-left corner of the screen. The list of names you have entered should still remain even after you leave this menu and come back to it.

Practising names

Once you click start, the program should show the Practice Recording menu:

Figure 11: Practice recording menu

This menu goes through all the names in your selected list of names and allows you to compare the database recording, your previous attempts, and your current attempt, in order to practise the name.

        To record an attempt, simply follow the same steps as with the Recording menu. This layout should be very familiar to you.

        To play and delete a previous attempt, simply follow the same steps as with the Listen menu. This layout should also be very familiar to you.

Once a new attempt has been recorded, it should show up in the list, with an index number one higher than the previous one (1, 2, 3 etc).

Apart from the previously mentioned functions, there are a few new components to this menu:

1)     The Compare button allows you to compare between the current recording and the database recording, by playing both of them one after the other repeatedly.

2)     The Database Play button plays the concatenated version of the name which was created by combining different database name files.

3)     The Next button allows the user to go to the next name to practice. You dont necessarily have to record a new attempt to proceed, but you can just have a go.

4)     You can check which name youre up to by looking at the Name Number Status label. This should give you a nice gauge of how many recordings you have left to go.

5)     You can also check your mic level here to troubleshoot any microphone problem you may have while recording.

6)     As always, you can press the Back button to return to the previous menu, in this case, it would be the Practice Menu.

After finishing the list of recordings and getting to the last recording, the Next button should convert into the Finish button.

Figure 12: Finish button

Clicking on this button should bring up a dialog to say that youve finished the exercise! Congratulations! You also earn some LipCoinsTM which can be used to purchase items from the Shop menu.

Figure 13: Practice finished dialog

 

Clicking the Thanks button on the dialog should bring you back to the Main menu. To double-check that your attempts are saved, we can go into the Listen menu again and check under the Attempted Names tab.

Figure 14: Listen menu, Attempts tab after practising

And here we can see that the recordings have saved successfully, and you can either listen to or delete them at will.

Finally, lets return to the Main menu and open the Shop menu:

Figure 15: Main menu to shop

Shop Menu

Figure 16: Shop menu

The Shop menu is the place to buy different upgrades and packages for NameSayer. Using the application will award you with LipCoinsTM, which you can use for purchasing different items in the shop.

You can earn LipCoinsTM by:

        Completing practices of names.

        Recording a new name into the database.

        Disliking a name.

        Launching the program.

Once you have enough LipCoinsTM, you may purchase any item by clicking on it and confirming your purchase. If an item is purchased, it can be activated and deactivated at will, by clicking on it.

Figure 17: Different statuses of shop items

You can see the different statuses of items in the figure above.
